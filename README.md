## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Home page URL: http://localhost:3000/home

Open the application home page

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
